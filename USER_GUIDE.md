# FinDesk — Короткий посібник користувача

## Основні можливості
- **Панель**: швидкі метрики доходів/витрат, міні-графік тренду, останні транзакції.
- **Транзакції**: пошук, фільтр за категоріями, контекстне меню для зміни категорії.
- **Бюджети**: створення лімітів за категоріями, попередження при досягненні порогів.
- **Рекурентні транзакції**: автоматичне створення повторюваних платежів за розкладом.
- **Аналітика**: тренди, прогноз балансу, аномалії, розбивка за категоріями.
- **Експорт**: CSV/Excel/PDF з урахуванням фільтрів та вибраних колонок.

## Швидкий старт
1. Запустіть застосунок і відкрийте вкладку **Джерела даних** для підключення банківських API або імпорту CSV.
2. Додайте транзакції через імпорт або вручну у вкладці **Транзакції**.
3. Налаштуйте бюджети у **Бюджети** та увімкніть рекурентні платежі за потреби.
4. Перейдіть до **Аналітика**, оберіть період і перегляньте тренди, прогноз та категорії.
5. Використовуйте кнопку **Експорт** у звітах для збереження CSV/Excel/PDF.

## Темна тема
- Основний фон: `#0B1220`, картки: `#0F172A` / `#111827`, текст: `#E5E7EB`.
- Всі ключові екрани оновлені на темну палітру з акцентами LiveCharts/OxyPlot.

## Поради
- Для коректного прогнозу бажано мати мінімум 2–3 місяці історичних даних.
- Якщо транзакції не відносяться до категорії, увімкніть авто-категоризацію або використайте контекстне меню в таблиці.
- Експорт великих звітів відбувається асинхронно; перевіряйте готовий файл у цільовій папці.
